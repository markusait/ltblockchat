Welcome to Tendermint Chat based on Lotion

Each message you send is included in a block and you can checkout the blockchai'n's state from either of these 3 nodes:

  • node1 (you are currently connected to this node over a socket connection)
  • node2
  • node3

This app acts as a lite client and can be run from different tabs. Note that you have to generate Keys first before sending encrypted messages.
